# Schuhe Marketplace

A modern React + Node.js marketplace for buying and selling used shoes.

## Project Structure

```
schuhe-marketplace/
├── frontend/          # React web application
├── backend/           # Node.js API server
├── .gitignore         # Git ignore file
└── README.md          # This file
```

## Quick Start

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

### Backend Setup
```bash
cd backend
npm install
npm start
```

## Technologies

- **Frontend:** React, TypeScript, CSS
- **Backend:** Node.js, Express, MongoDB (planned)
- **Deployment:** (To be configured)

## Development

Each directory (frontend/backend) contains its own package.json and development configuration.
